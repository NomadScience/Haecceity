Examples

All of the examples work in basically the same way. Simply create a new instance (with optional settings) and mount it to an element.

The canvas background for all visualizations are transparent by default.

DFT Cube

The DftCube can be created with the following options:

const cube = new haecceity.DftCube({
  count: 100, // default 100
  speed: 1, // default 1
  cubeBaseColor: '#00f', // default #00f
  cubeGlowColor: 'somethingorother', // default some kind of crazy random hardcoded color
})

Periodic Table

The periodic table is designed to take in multiple color-schemes and to switch between three layouts.

Out of the box, it includes CPK color assignment and two types of coloring by element group, one associated with the Stowe layout

It can also switch between three separate layouts:

  • Mendaleev
  • Flattened Stowe
  • 3D Stowe

    • In Stowe mode, there is an optional overlay to explain the Stowe layout
